# DOCSNAP: Summarization of Text and Image Captioning

## Overview
DOCSNAP is an AI-based application that generates summaries from text and creates captions for images. It helps users quickly understand the content of documents and images.

## Features
- Text Summarization
- Image Caption Generation
- Easy-to-use Interface
- Fast Processing

## Technologies Used
- Python
- Artificial Intelligence
- Natural Language Processing (NLP)

## Project Structure

DOCSNAP/

├── app.py          # Main application file

└── README.md       # Project documentation

## How to Run

1. Clone the repository.
2. Install the required Python libraries.
3. Run:

python app.py

4. Open the application in your browser.
